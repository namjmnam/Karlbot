# Karlbot
카카오톡 자동응답을 위한 봇입니다.

## 설치
안드로이드 기기에 설치한 뒤 메인 UI를 통해 권한을 부여해주세요.
안드로이드 버전 7.0 이하의 기기에서는 Wear OS를 설치해야 합니다.
```
https://play.google.com/store/apps/details?id=com.google.android.wearable.app
```

## 등록된 명령어

/start : 이 메시지 출력\
/say : 말시키기\
/game : 준비중\
/quiz : 준비중\
/dice : 랜덤 주사위\
/naver : 네이버 검색\
/ddg : DuckDuckGo 검색\
/save : 데이터 저장\
/load : 데이터 불러오기\
/reset : 저장된 변수 리셋\
/troll : ???
